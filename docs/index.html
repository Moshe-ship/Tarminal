What's inside
‏
RTL BiDi Rendering
Core Text overlay re-renders Arabic and Hebrew lines right-to-left with full Unicode BiDi Algorithm support.
«
Connected Letters
Arabic letters connect properly. Initial, medial, final forms. Not broken rectangles like every other terminal.
□
Tab Groups NEW
Color-coded tab groups. Right-click to assign colors. Drag tabs to reorder. Cmd+1-9 to switch.
↓
Drag & Drop NEW
Drop files and images onto the terminal. Paths are auto-escaped and pasted at the cursor.
¶
Find in Scrollback NEW
Cmd+F to search terminal output. Cmd+G / Cmd+Shift+G to navigate matches.
§
5 Themes
Tarminal Dark, Light, Dracula, Nord, and Sarab. Window transparency with vibrancy. All ANSI colors.
¸
All Settings Wired
Cursor style, blink, scrollback buffer, bell control, Option-as-Meta, BiDi mode. Every toggle does what it says.
>
Full Terminal
zsh, bash, fish. VT100/xterm emulation via SwiftTerm. Clickable URLs. Sixel & iTerm2 image protocol support.
∅
Open Source
MIT licensed. Swift + SwiftUI. Built on SwiftTerm. Fork it, extend it, make it yours.
